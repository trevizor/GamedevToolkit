Plan Maker Help

Basics

  • Pick a tool, then click-drag on the canvas to place geometry.
  • Use the right Levels panel to add/reorder/remove levels and toggle Onion View.
  • Use Save Plan to generate JSON and Load Plan to restore it.
  • Use Export Model for GLB output or Preview 3D to inspect geometry.

Selection and Editing

  • Select/Move tool: click to select, drag to move, drag in empty area for marquee select.
  • Use Shift to add to selection and Ctrl to remove from selection.
  • Press Delete/Backspace or X to remove selected items.
  • With slabs/ramps selected, Q/E rotates selection.

Shortcuts

  • W cycles wall tools, D cycles door/window/cut, F cycles slab tools, R cycles ramp tools, S select, X erase/delete.
  • Ctrl/Cmd+Z undo, Ctrl/Cmd+Y redo, Ctrl/Cmd+A select all visible items.
  • Middle mouse button pans the canvas, mouse wheel zooms.

Unity Usage

  1. In this editor, click Save Plan and keep the generated JSON file.
  2. In Unity, add the script to a GameObject and assign your prefabs plus the JSON TextAsset.
  3. Click Build From Assigned JSON in the component context menu, or use the inspector button provided by the editor script.
  4. The importer spawns walls, doors/windows openings, slabs, ramps, half ramps, and top half ramps according to the JSON.
